🧪 Evaluation Methodology

Leave-One-Out Protocol: For each user, items rated ≥ 4.0 were held out as test items. The model was asked to rank all un-interacted items. Evaluation metrics were computed at K=10.

Precision@10: Fraction of top-10 recommended items that are relevant.
Recall@10: Fraction of all relevant items captured in top-10 recommendations.
NDCG@10: Normalized Discounted Cumulative Gain — rewards relevant items appearing higher in the ranked list.

👥 Collaborative Filtering

Uses SVD (Singular Value Decomposition) to decompose the user-item rating matrix into latent factor matrices. Captures hidden user-item interaction patterns.

R ≈ U · Σ · Vᵀ
rank = 40 latent factors
variance explained: 56.9%

📄 Content-Based

Builds TF-IDF vectors from product category, brand, and tags. Computes cosine similarity between items. Recommends products similar to ones the user liked.

sim(a,b) = cos(v_a, v_b)
features: category + brand
vocab: 200 TF-IDF terms

🔀 Hybrid Model

Linearly combines normalized CF and CB scores. The α weight (0.70) favors collaborative filtering while content-based handles cold-start scenarios.

score = 0.7·CF + 0.3·CB
both scores normalized [0,1]
rated items excluded
